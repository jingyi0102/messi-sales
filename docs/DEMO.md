# Demo Walkthrough

Use this sequence for a 90-second portfolio demonstration.

## 1. Admin

1. Sign in with `admin / admin123`.
2. Show the monthly dashboard and salesperson performance.
3. Open **Admin → Accounts**.
4. Create one salesman account and one driver account.
5. Create an order, assign the salesman and driver, and show the live balance
   and profit calculations.

## 2. Salesman

1. Sign out and sign in with the salesman account.
2. Show that only the assigned order appears.
3. Point out that company profit, costing, admin controls, editing, deletion,
   and payment controls are unavailable.

## 3. Driver

1. Sign out and sign in with the driver account.
2. Open **Delivery** and show only the assigned job.
3. Select **Start**, then **Complete**.
4. Open **Completed** and show the delivered order.

## Closing

Explain that the same TypeScript codebase runs on Web, iOS, and Android, with
local SQLite persistence. The next production step would be cloud
authentication and multi-device synchronization.
